<task type="auto">
  <name>Task 1: Install deps + buat semua Drizzle files</name>
  <files>package.json, drizzle.config.ts, db/schema.ts, db/index.ts, scripts/migrate-from-supabase.ts, .gitignore</files>
  <action>
    Step 1 — Install dependencies:
    ```
    npm install drizzle-orm @neondatabase/serverless
    npm install -D drizzle-kit dotenv tsx
    ```

    Step 2 — Buat `db/schema.ts`:
    Drizzle schema yang mirror persis db_schema.sql:
    - Tabel `transactions`: id (uuid, primaryKey, defaultRandom()), created_at (bigint mode:'number', notNull), date (date, notNull), description (text, notNull), type (text, notNull), amount (numeric precision:15 scale:2, notNull, default:'0')
    - Tabel `appUsers` → nama SQL tetap 'app_users': id (uuid pk), username (text, unique, notNull), password (text, notNull), name (text, notNull), role (text, notNull)
    - Tambah indexes: idx_transactions_date (date), idx_transactions_type (type), idx_transactions_created_at (created_at), idx_app_users_username (username)
    - Gunakan named exports: `export const transactions = pgTable(...)` dan `export const appUsers = pgTable(...)`

    Step 3 — Buat `db/index.ts`:
    ```ts
    import { neon } from '@neondatabase/serverless';
    import { drizzle } from 'drizzle-orm/neon-http';
    import * as schema from './schema';

    const sql = neon(process.env.DATABASE_URL!);
    export const db = drizzle(sql, { schema });
    ```

    Step 4 — Buat `drizzle.config.ts` di root:
    ```ts
    import { defineConfig } from 'drizzle-kit';
    import { config } from 'dotenv';

    config({ path: '.env.local' });

    export default defineConfig({
      schema: './db/schema.ts',
      out: './drizzle',
      dialect: 'postgresql',
      dbCredentials: {
        url: process.env.DATABASE_URL!,
      },
    });
    ```

    Step 5 — Buat `scripts/migrate-from-supabase.ts`:
    - Load .env.local via dotenv config({ path: '.env.local' })
    - Buat Supabase client dengan kredensial dari supabaseClient.ts (hardcoded URL + KEY sudah ada di sana)
    - Import db dari '../db/index' dan { transactions, appUsers } dari '../db/schema'
    - Fetch semua baris: supabase.from('transactions').select('*') — ambil semua tanpa limit
    - Fetch semua baris: supabase.from('app_users').select('*')
    - Insert ke NeonDB via db.insert(transactions).values(txData.map(...)) — pastikan created_at di-cast ke Number()
    - Insert ke NeonDB via db.insert(appUsers).values(userData)
    - Console.log jumlah row tiap tabel
    - Jika Supabase error, throw dan jangan lanjut ke insert NeonDB

    Step 6 — Update `.gitignore`:
    Tambahkan baris `.env.local` jika belum ada.

    PENTING: db/index.ts adalah server-only. Jangan import dari komponen React atau file frontend manapun.
  </action>
  <verify>
    Jalankan `npm run build` — harus sukses tanpa error baru.
    Jalankan `npx tsc --noEmit` — tidak ada type error di file baru.

  </verify>
  <done>AC-3 satisfied: schema.ts merefleksikan db_schema.sql; semua deps terinstall; semua file terbuat</done>
