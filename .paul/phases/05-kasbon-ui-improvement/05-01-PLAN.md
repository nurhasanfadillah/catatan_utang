- `App.tsx` — dashboard 3 StatsCards, filter summary label "Halaman ini:"

## AC-1: Label Kolom Kasbon Benar
```gherkin
Given user membuka tab "Data Kasbon" di desktop (≥ md breakpoint)
When melihat header tabel TransactionList
Then kolom keempat menampilkan "Kasbon (Keluar)" (bukan "Diterima (Keluar)")
  Dan kolom ketiga tetap "Tagihan (Masuk)"
```

## AC-2: Dashboard Menampilkan 3 Stats
```gherkin
Given user login dan berada di tab "Dashboard"
When halaman dashboard render
Then terlihat 3 StatsCard:
  - "Saldo Saat Ini" dengan variant="balance" (gradient indigo-violet)
  - "Total Tagihan (Halaman Ini)" dengan variant="income" (emerald border-left)
  - "Total Kasbon (Halaman Ini)" dengan variant="expense" (rose border-left)
  Dan layout menggunakan grid 1 kolom di mobile, 3 kolom di sm+
```

## AC-3: Warna Amount Konsisten di Dark Mode
```gherkin
Given user melihat TransactionList di dark mode
When melihat nominal transaksi (di mobile card MAUPUN desktop table)
Then warna INCOME menggunakan text-emerald-400 (bukan text-emerald-600)
  Dan warna EXPENSE menggunakan text-rose-400 (bukan text-rose-600)
  Dan konsisten dengan warna icon badge di mobile card (sudah text-emerald-400/rose-400)
```

## AC-4: Filter Summary Ada Label Konteks
```gherkin
Given user mengaktifkan filter (type atau tanggal)
When filter summary badges muncul di bawah filter bar
Then setiap badge memiliki prefix label "Halaman ini:" atau keterangan "data halaman ini"
  Sehingga jelas bahwa angka yang ditampilkan bukan total global
```

## AC-5: Empty State Informatif
```gherkin
Given tidak ada transaksi yang tampil (data kosong atau filter tidak menemukan data)
When TransactionList render
Then tampil icon Inbox (dari lucide-react) di atas teks
  Dan teks utama: "Tidak ada data ditemukan."
  Dan teks sekunder: "Coba ubah filter atau tambah data baru."
```
