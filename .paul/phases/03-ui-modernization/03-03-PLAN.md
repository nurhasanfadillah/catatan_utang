<objective>
## Goal
Redesign TransactionList, TransactionForm, ExportModal, ConfirmationModal, dan App.tsx ke dark fintech palette — mengganti semua sisa `gray-*` dan `brand-*` references dengan `slate-*` dan `indigo-*` sesuai design system yang sudah di-establish di plan 03-02.

## Purpose
Phase 03 tidak akan selesai selama komponen-komponen ini masih pakai `bg-gray-*` dan `text-brand-*` — tampilan akan inkonsisten antara sidebar/header (sudah dark) dengan konten utama (masih gray). Plan ini menutup loop visual.

## Output
5 file dimodifikasi. Setelah selesai, seluruh UI konsisten di slate-900/800 surfaces + indigo-500 accents + gradient CTAs.
</objective>

## Design System yang Sudah Di-establish (03-02)
Pattern yang harus direplikasi ke semua komponen berikut:
- **Surfaces:** `bg-slate-900` (page) / `bg-slate-800` (cards, modals) / `bg-slate-700/50` (inputs)
- **Borders:** `border-slate-700` (standard) / `border-slate-700/50` (subtle)
- **Text:** `text-slate-100` (primary) / `text-slate-300` (secondary) / `text-slate-400` (muted)
- **Accent:** `text-indigo-400` / `bg-indigo-500` / `bg-indigo-600`
- **CTA Primary:** `bg-gradient-to-r from-indigo-500 to-violet-600 hover:from-indigo-600 hover:to-violet-700`
- **Semantic (tetap):** emerald untuk income/Tagihan, rose untuk expense/Kasbon/danger
- **Focus ring:** `focus:ring-indigo-500` (ganti dari `focus:ring-brand-500`)
